PENGERTIAN LINUX

Linux adalah sistem operasi berbasis open source yang memungkinkan siapa saja untuk menggunakan, memodifikasi, dan mendistribusikannya secara bebas. Sistem ini berfungsi sebagai dasar bagi berbagai perangkat, serupa dengan peran sistem operasi lain, tetapi memiliki fleksibilitas tinggi karena kode sumbernya terbuka.

Riwayat Singkat Linux:

  • Dikembangkan oleh Linus Torvalds pada 1991 sebagai proyek pribadi.
  • Kernel Linux digabung dengan perangkat lunak GNU sehingga menjadi sistem operasi lengkap.
  • Muncul berbagai distribusi seperti Debian, Red Hat, Slackware, Ubuntu, dan Arch Linux.
  • Digunakan secara luas pada server, superkomputer, IoT, Android, dan desktop.

LINUX POPULER

Arch Linux

Arch Linux

Arch Linux adalah sistem operasi yang ringan dan minimalis, berpusat pada pengguna sehingga dapat menyesuaikan banyak aspek sistem secara bebas.

Linux History

Pada 11 Maret 2002, Judd Vinet mulai mengembangkan Arch Linux terinspirasi dari Slackware dan CRUX. Ia kemudian menulis Pacman, pengelola paket yang menjadi ciri khas Arch Linux.

Package Manager

Arch menggunakan Pacman untuk menginstal, menghapus, dan memperbarui paket. Banyak GUI tersedia seperti Pamac dan Octopi meskipun Pacman sendiri berbasis CLI.

Ubuntu

Ubuntu adalah distribusi Linux populer dan ramah pengguna berbasis Debian. Sering direkomendasikan untuk pemula.

Based on Debian

Ubuntu menggunakan sistem manajemen paket Debian (APT). Pertama dirilis tahun 2004 oleh Canonical Ltd.

Desktop Environment

Desktop default-nya adalah GNOME. Varian resmi lain mencakup Kubuntu (KDE), Xubuntu (XFCE), dan Ubuntu MATE.

Linux Mint

Linux Mint adalah distribusi yang elegan dan mudah digunakan, berbasis Ubuntu dan Debian.

Defaults

Mint menawarkan pengalaman desktop klasik dengan dukungan multimedia dan tools bawaan.

Desktop Environment

Edisi utama menggunakan Cinnamon, terkenal dengan tampilan tradisional dan sangat dapat dikustomisasi.

Kali Linux

Kali Linux adalah distribusi berbasis Debian yang dirancang untuk forensik digital dan penetration testing.

Key Use

Dikembangkan oleh Offensive Security, berisi ratusan tools untuk keamanan seperti pentesting dan reverse engineering.

Big Tools

Termasuk Metasploit, Nmap, Wireshark, dan banyak alat lainnya.
