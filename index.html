PENGERTIAN LINUX

Linux adalah sistem operasi berbasis open source yang memungkinkan siapa saja untuk menggunakan, memodifikasi, dan mendistribusikannya secara bebas. Sistem ini berfungsi sebagai dasar bagi berbagai perangkat, serupa dengan peran sistem operasi lain, tetapi memiliki fleksibilitas tinggi karena kode sumbernya terbuka.

Riwayat singkat Linux:

  • Linux dikembangkan oleh Linus Torvalds pada tahun 1991 sebagai proyek pribadi ketika ia masih menjadi mahasiswa di Finlandia.
  • Ia merancang sebuah kernel sistem operasi yang dapat digunakan secara gratis dan dikembangkan bersama komunitas.
  • Kernel tersebut kemudian dipadukan dengan berbagai perangkat lunak dari proyek GNU sehingga membentuk sistem operasi yang lengkap.
  • Setelah itu muncul berbagai distribusi seperti Debian, Red Hat, Slackware, Ubuntu, dan Arch Linux.
  • Saat ini Linux digunakan secara luas pada server, superkomputer, perangkat IoT, ponsel berbasis Android, serta oleh para pengguna komputer sebagai sistem operasi harian.

LINUX POPULER

Arch Linux

Arch Linux

Arch Linux adalah sistem operasi terdistribusi yang ringan, sederhana, dan modern dengan sifat minimal dan berpusat pada pengguna untuk menyesuaikan dan mengontrol sistem dalam banyak hal, sehingga pengguna dapat menyesuaikan berbagai bagian sistem sambil mendapatkan manfaat dari sistem yang terintegrasi.

Linux History

Pada 11 Maret 2002, Judd Vinet terinspirasi oleh kesederhanaan dan keeleganan yang dimiliki oleh Slackware, distribusi Linux di Polandia dan CRUX, ia pun mulai mengembangkan Arch Linux. Namun karena kurangnya program pengelola paket menjadi masalah utama, ia memutuskan untuk menulis pengelola paket khusus untuk Arch Linux yang disebut Pacman untuk mengontrol dan memeriksa instalasi, penghapusan, dan pembaruan paket perangkat lunak secara otomatis.

Package manager

Arch Linux menggunakan Pacman (Package Manager), digunakan untuk menginstal, menghapus, dan memperbarui paket perangkat lunak. Ia menangani dependensi, melakukan sinkronisasi dengan repositori jarak jauh, dan dapat digunakan untuk mengelola paket repositori resmi dan paket yang dibuat pengguna. Meskipun tidak ada manajer paket grafis bawaan, banyak alat GUI yang mudah digunakan tersedia untuk Arch Linux, seperti Pamac dan Octopi.

Ubuntu

Ubuntu is a popular, user-friendly, and community-driven Linux distribution based on Debian. It's often recommended for beginners.

Based on Debian

Ubuntu is primarily based on Debian's architecture and package management system (APT). It was created by Canonical Ltd. and first released in 2004.

Desktop Environment

The default desktop environment for Ubuntu is **GNOME**, but various official flavors use different environments like KDE (Kubuntu), XFCE (Xubuntu), and MATE (Ubuntu MATE).

Linux Mint

Linux Mint is an elegant, easy-to-use, and community-driven operating system based on Ubuntu and Debian.

DEFAULTS

Mint offers a classical desktop experience and comes bundled with multimedia support and essential tools, making it very popular for users transitioning from Windows or macOS.

Desktop Environment

Its main edition uses the **Cinnamon** desktop environment, which is known for its traditional and highly customizable interface.

Kali Linux

Kali Linux is a Debian-derived distribution designed for digital forensics and penetration testing.

Key Use

It is maintained and funded by Offensive Security and contains hundreds of tools for various information security tasks, such as penetration testing, security research, and reverse engineering.

Big Tool

Tools include Metasploit, Nmap, Wireshark, and many others, pre-installed and ready to use for security professionals and ethical hackers.
